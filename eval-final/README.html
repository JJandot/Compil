3 - Ajout de l'instruction incr

On désire ajouter à notre langage l'instruction incr qui effectue une incrémentation par un d'une variable. La syntaxe de cette nouvelle instruction est simple : incr var où var est une variable simple ou une case de tableau.

Modifiez l'analyseur lexical, l'analyseur syntaxique, la structure de l'arbre abstrait, et la génération de code afin de prendre en compte cette nouvelle instruction.
